-----------------------------------Sharing Device Identification on Images from Social Media Platforms----------------------------------

Features.txt - This file contains a list of 9 AC components of dequantized DCT
Labele.txt - This file contains the labels for above file features

Features_all.txt - This file contains a list of all AC components of dequantized DCT
Labele_all.txt - This file contains the labels for above file features

Above files are created to save time of extracting DCT features again and again as it is a time consuming task
